# 📥 MediaFire Folder Downloader

A simple Python script that downloads all files from a public MediaFire folder.

## 🚀 Features

- Downloads all files inside a MediaFire folder
- Extracts real download links (prevents corrupted files)
- Saves files with correct filenames
- Lightweight and easy to use

---

## 📦 Requirements

- Python 3.8+
- requests
- beautifulsoup4

Install dependencies:

```bash
pip install -r requirements.txt
